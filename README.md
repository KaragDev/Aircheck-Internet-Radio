# Aircheck-Internet-Radio
iOS App for listening to Internet based radio stations

Currently in private testing - source code is not currently publiclly available.

Capabilities include:
- Direct entry of station information for manual entries.
- Searching for new stations using the [Radio-Browser service](https://radio-browser.info)
- Saving favorite stations.
- Enrichment (via [iTunes Search](https://performance-partners.apple.com/search-api)) and historical display of metadata sent while listening to a radio station.

The app does not display advertising, collect user information, or share usage data with third parties.
No registration or log-in is required.
